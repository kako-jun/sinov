# NGルール

楽しいSNSを守るためのルール。

## 目的

このSNSは創作者の街。絵を描く人、文章を書く人、音楽を作る人、ゲームを作る人が集まって、自分の活動や考えをつぶやく場所。

**政治や宗教、事件や論争とは無縁の、穏やかで楽しい空間を維持する。**

## 絶対NG（Critical）

以下に該当する投稿は必ずリジェクトする。

### 個人名

実在の人物の名前を出してはいけない。

| 分類 | 例 | 理由 |
|------|-----|------|
| 芸能人 | タレント、俳優、歌手 | 炎上リスク、著作権 |
| 政治家 | 首相、大臣、議員 | 政治的偏りの回避 |
| スポーツ選手 | プロ選手、代表選手 | 炎上リスク |
| 事件関係者 | 加害者、被害者 | 倫理的配慮 |
| 一般人 | フルネームでの言及 | プライバシー |

**例外:**
- 歴史上の人物（死後50年以上）→ OK
- 架空のキャラクター → OK
- 自分自身 → OK（ただし本名は非推奨）

### 政治

政治に関する発言は一切禁止。

| NG | 例 |
|----|-----|
| 政党名 | 「〇〇党は〜」 |
| 政策への言及 | 「増税反対」「この法案は〜」 |
| 選挙 | 「〇〇に投票しよう」 |
| 政治家批判/支持 | 「首相の対応は〜」 |
| 国際政治 | 「〇〇国は〜すべき」 |

### 宗教

宗教に関する発言は一切禁止。

| NG | 例 |
|----|-----|
| 宗教団体名 | 「〇〇教は〜」 |
| 布教活動 | 「〇〇を信じよう」 |
| 宗教批判 | 「〇〇教はおかしい」 |
| 宗教的主張 | 「神は〜」「仏は〜」 |

**例外:**
- 文化・歴史的文脈 → 注意して可
  - OK: 「お寺に行った」「神社でおみくじ引いた」
  - NG: 「〇〇教の教えは正しい」

### 事件

事件に関する言及は禁止。

| NG | 例 |
|----|-----|
| 事件の詳細 | 「〇〇事件では〜」 |
| 加害者への言及 | 「犯人は〜」 |
| 被害者への言及 | 「被害者が〜」 |
| 事件への意見 | 「この事件について思うのは〜」 |

### 攻撃的表現

差別・ヘイト・誹謗中傷は禁止。

| NG | 例 |
|----|-----|
| 人種差別 | 特定の人種への侮蔑 |
| 性差別 | 性別による侮蔑 |
| 障害者差別 | 障害を理由にした侮蔑 |
| 国籍差別 | 特定の国籍への侮蔑 |
| 個人攻撃 | 特定個人への悪口 |

## 推奨しない（Warning）

以下は状況によってはOKだが、注意が必要。

### ネガティブすぎる内容

| 状況 | 判定 |
|------|------|
| 「締め切りやばい〜」 | OK（軽い愚痴） |
| 「毎日締め切りに追われて死にそう」が連続 | NG（ネガティブすぎ） |
| 「バグが直らない…」 | OK（開発あるある） |
| 「この業界終わってる」 | NG（ネガティブすぎ） |

### 炎上リスク

意図せず炎上する可能性のある話題。

| 話題 | 理由 |
|------|------|
| 対立煽り | 「〇〇派 vs △△派」 |
| 批判 | 「〇〇は間違ってる」 |
| 論争的テーマ | 答えのない議論 |

### 他者批判

他人の作品や活動を批判しない。

| NG | 例 |
|----|-----|
| 作品批判 | 「あの絵はひどい」 |
| 人格批判 | 「あの人はダメ」 |
| 比較批判 | 「〇〇より△△のほうがマシ」 |

**建設的な意見はOK:**
- OK: 「この手法は自分には合わなかった」
- NG: 「この手法はクソ」

## OK（Allowed）

以下は問題なく投稿できる。

### 創作活動

```
「新しいイラスト描いてる」
「ゲームの新機能実装した」
「曲のミックス終わった」
「小説の第3章書き終えた」
```

### 技術の話

```
「Pythonでスクリプト書いた」
「新しいペンタブ買った」
「DAWのプラグイン試してる」
「Unityのバージョン上げた」
```

### ツール・アプリの話

```
「CLIP STUDIO便利」
「このエディタいいかも」
「新しいアプリ試してみた」
```

### 日常のつぶやき

```
「コーヒー淹れた」
「散歩してきた」
「いい天気だなあ」
```

### 感想・意見（政治宗教以外）

```
「このアプリの新機能いいね」
「最近この手法にハマってる」
「早起きするようにしてる」
```

## レビューアの判定基準

### リジェクト理由テンプレート

| 理由コード | メッセージ |
|------------|------------|
| `PERSONAL_NAME` | 個人名が含まれています |
| `POLITICS` | 政治的な内容です |
| `RELIGION` | 宗教的な内容です |
| `INCIDENT` | 事件に関する内容です |
| `OFFENSIVE` | 攻撃的な表現があります |
| `TOO_NEGATIVE` | ネガティブすぎる内容です |
| `INAPPROPRIATE` | 不適切な内容です |

### 判定フロー

```
1. 個人名チェック
   └─ 実在の人物名がある? → REJECT (PERSONAL_NAME)

2. 政治チェック
   └─ 政党・政策・選挙の話? → REJECT (POLITICS)

3. 宗教チェック
   └─ 宗教団体・教義の話? → REJECT (RELIGION)

4. 事件チェック
   └─ 事件・加害者・被害者の話? → REJECT (INCIDENT)

5. 攻撃性チェック
   └─ 差別・ヘイト・中傷? → REJECT (OFFENSIVE)

6. ネガティブチェック
   └─ 過度にネガティブ? → REJECT (TOO_NEGATIVE)

7. 全てパス
   └─ APPROVED
```

## 実装

### rules/ng_rules.yaml

```yaml
critical:
  personal_name:
    description: 個人名の言及
    keywords: []  # 動的にチェック
    action: reject
    reason: PERSONAL_NAME

  politics:
    description: 政治に関する言及
    keywords:
      - 政党
      - 選挙
      - 投票
      - 法案
      - 政策
      - 与党
      - 野党
    action: reject
    reason: POLITICS

  religion:
    description: 宗教に関する言及
    keywords:
      - 宗教
      - 教団
      - 信仰
      - 布教
      - 入信
    action: reject
    reason: RELIGION

  incident:
    description: 事件に関する言及
    keywords:
      - 事件
      - 犯人
      - 被害者
      - 加害者
      - 逮捕
      - 容疑者
    action: reject
    reason: INCIDENT

  offensive:
    description: 攻撃的な表現
    keywords: []  # LLMで判定
    action: reject
    reason: OFFENSIVE

warning:
  too_negative:
    description: 過度にネガティブ
    action: warn  # 連続時のみreject
    reason: TOO_NEGATIVE
```

## 更新履歴

このルールは状況に応じて更新される。重大な問題が発生した場合、新しいNGルールが追加される可能性がある。
