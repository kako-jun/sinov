# パラメータ詳細仕様

**注意: このドキュメントの内容は実装済みです。**

実装済みの仕様は `docs/models.md` も参照。

## 追加の性格パラメータ（3個）

現在の11個に追加で検討しているパラメータ。

| パラメータ | 説明 | 低い | 高い |
|-----------|------|------|------|
| `event_enthusiasm` | イベント熱狂度 | 興味なし | 大好き |
| `contrarian` | 逆張り度 | 素直 | ひねくれ者 |
| `eccentricity` | 不思議ちゃん度 | 普通の人 | 電波系 |

### event_enthusiasm（イベント熱狂度）

季節イベントへの反応確率に影響。

| 値 | 発言の特徴 |
|----|-----------|
| 0.1〜0.3 | 「世間はクリスマスらしい」「正月？普通に仕事」 |
| 0.4〜0.6 | 「クリスマスか〜」「正月はゆっくりする」 |
| 0.7〜0.9 | 「クリスマスだー！！」「正月休み最高！」 |

### contrarian（逆張り度）

流行への逆張り確率、皮肉っぽい発言に影響。

| 値 | 発言の特徴 |
|----|-----------|
| 0.1〜0.3 | 素直に流行に乗る |
| 0.4〜0.6 | たまに違う視点を出す |
| 0.7〜0.9 | 「みんながいいって言うけど、実際どうなの」 |

### eccentricity（不思議ちゃん度）

発言の飛躍度、独自の世界観に影響。mental_health（変動）とは別物で、固定の性格特性。

| 値 | 発言の特徴 |
|----|-----------|
| 0.1〜0.3 | 普通の会話、文脈に沿う |
| 0.4〜0.6 | たまに独特の視点、少し飛躍 |
| 0.7〜0.9 | 「今日の空は三角形の味がする」 |

## 追加の状態パラメータ（6個）

現在の `mood` に追加で検討しているパラメータ。

| パラメータ | 説明 | 変化要因 |
|-----------|------|----------|
| `energy` | エネルギー | 時間帯、活動量 |
| `focus` | 集中度 | 連作中は高い |
| `motivation` | モチベ | 反応、達成感 |
| `fatigue` | 疲労度 | 投稿数、深夜作業 |
| `excitement` | 興奮度 | イベント、良い反応 |
| `mental_health` | メンタル | 反応なし、否定、孤立 |

### 疲労度（fatigue）

| 値 | 発言の特徴 |
|----|-----------|
| 0.0〜0.3 | 元気、普通の投稿 |
| 0.4〜0.6 | 「ちょっと疲れた」「休憩」 |
| 0.7〜0.9 | 「疲れた…」「もう寝る」投稿頻度↓ |

### 興奮度（excitement）

| 値 | 発言の特徴 |
|----|-----------|
| 0.0〜0.3 | 落ち着いた投稿 |
| 0.4〜0.6 | 少しテンション高め |
| 0.7〜0.9 | 「！！」「やばい！」連投 |

### メンタル（mental_health）

| 値 | 発言の特徴 |
|----|-----------|
| 0.7〜1.0 | 普通の投稿、健康的 |
| 0.5〜0.6 | 少しネガティブ |
| 0.3〜0.4 | 「なんかつらい」頻度↓ |
| 0.1〜0.2 | 数日投稿なし |

## 活動時間帯の詳細設定

現在の `active_hours` に追加で検討している詳細設定。

### hourly_weight

時間帯ごとの活動確率を細かく指定。

```yaml
hourly_weight:
  0: 0.1   # 深夜は低確率
  10: 0.8  # 午前は高め
  22: 1.0  # ピーク
```

### chronotype（クロノタイプ）

生物学的な活動リズム。

| タイプ | 説明 | 活動時間 |
|--------|------|----------|
| lark | 朝型 | 5:00-21:00 |
| owl | 夜型 | 12:00-4:00 |
| intermediate | 中間型 | 7:00-23:00 |

### rhythm_stability（生活リズムの安定度）

```yaml
regular_life:
  rhythm_stability: 0.9  # 毎日同じ時間

irregular_life:
  rhythm_stability: 0.3  # 日によって変わる
```

### daily_schedule（生活スケジュール）

時間帯ごとの活動確率分布。

```yaml
daily_schedule:
  7:
    waking_up: 0.4
    commuting: 0.3
    breakfast: 0.2
  12:
    lunch: 0.6
    working: 0.3
  23:
    working: 0.4
    going_to_bed: 0.2
```

## 制作物システム

各住人が作っている架空の作品を管理。

### 基本構造

```yaml
creative_works:
  current:
    - name: 「星降る夜に」
      type: illustration_series
      progress: 0.6
      current_episode: 第3話
  completed:
    - name: 「夏の記憶」
      completed_at: "2024-12-15"
  planned:
    - name: 「冬のシリーズ」
```

### 作品タイプ

| 職業 | タイプ |
|------|--------|
| イラストレーター | illustration, illustration_series, fan_art |
| ゲーム開発者 | indie_game, game_jam, prototype |
| 作曲家 | single, album, bgm |
| 小説家 | novel, short_story, web_serial |

### 進捗による発言変化

| 進捗 | 発言例 |
|------|--------|
| 0〜20% | 「{name}、始めた」 |
| 20〜50% | 「{name}、やっと{progress}%」 |
| 50〜80% | 「{name}、あと少し」 |
| 100% | 「{name}、完成した！」 |

### 進捗の変化

```python
progress_delta = {
    "working": 0.02,      # 通常作業
    "focused": 0.05,      # 集中して作業
    "breakthrough": 0.1,  # ブレイクスルー
}
```

完成時は長期記憶に昇格。
