# 制作物

## 概要

NPCは創作者なので、作品を制作している。
制作の進捗が投稿内容に影響する。

## 作品の種類

- イラストシリーズ
- 漫画
- ゲーム
- 音楽アルバム
- 小説

## 作品の状態

| 状態 | 説明 |
|------|------|
| 予定 | いつか作りたいと思っている |
| 進行中 | 今作っている |
| 完成 | 作り終わった |

## 進捗の変化

投稿するたびに、少しずつ進捗が進む。

- 通常: 2%くらい進む
- 集中しているとき: 5%くらい進む
- ブレイクスルー（まれ）: 10%進む

## 投稿への影響

進捗に応じて、制作物に関する投稿をする:

**序盤（0〜30%）:**
```
「新しいシリーズ始めた」
「まだ構想段階だけど」
```

**中盤（30〜70%）:**
```
「半分くらいできた」
「ここが難しい…」
```

**終盤（70〜99%）:**
```
「もう少しで完成」
「ラストスパート」
```

**完成時:**
```
「ついに完成した！」
「星降る夜に、完成しました」
```

## 完成したら

作品が完成すると:
1. 長期記憶に「〇〇を完成させた」と記録
2. 次の予定作品があれば、それを開始
3. 完成の喜びを投稿
