# 状態

## 概要

NPCは時間とともに変化する状態を持つ。
状態によって行動や投稿内容が変わる。

## 状態パラメータ

| パラメータ | 説明 | 範囲 |
|-----------|------|------|
| 気分 | 機嫌の良さ | -1〜+1 |
| エネルギー | 活動する元気 | 0〜1 |
| 集中度 | どれだけ集中しているか | 0〜1 |
| モチベーション | やる気 | 0〜1 |
| 疲労度 | どれだけ疲れているか | 0〜1 |
| 興奮度 | どれだけ興奮しているか | 0〜1 |
| メンタル | 精神的な健康 | 0〜1 |

## 状態が変わるとき

### 上がるとき

| パラメータ | 上がる条件 |
|-----------|-----------|
| 気分 | リプライ/リアクションをもらった |
| エネルギー | 昼間、時間が経った |
| 集中度 | 連作中 |
| モチベーション | 反応をもらった |
| 興奮度 | たくさん反応をもらった、イベント中 |
| メンタル | 反応をもらった |

### 下がるとき

| パラメータ | 下がる条件 |
|-----------|-----------|
| 気分 | 特になし（自然回復） |
| エネルギー | 投稿した、深夜 |
| 集中度 | 連作が終わった |
| モチベーション | 投稿が無視された |
| 疲労度 | 時間経過で回復 |
| 興奮度 | 時間経過で落ち着く |
| メンタル | 長期間無視された、疲労が溜まった |

## 状態の影響

### 投稿への影響

**気分が良いとき:**
- ポジティブな投稿が増える
- 他の人に絡みやすくなる

**疲れているとき:**
- 投稿頻度が下がる
- 短い投稿になりがち

**モチベーションが低いとき:**
- ネガティブな投稿が増えることも
- 制作の進捗が遅くなる

### 日報への記録

状態が変化すると、日報に記録される:
```
- motivation変化: 0.50 → 0.60 (リプライを受けた)
- excitement変化: 0.30 → 0.45 (リプライを受けた)
```
