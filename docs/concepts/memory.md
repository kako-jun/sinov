# 記憶

## 記憶の種類

NPCは2種類の記憶を持つ:

### 短期記憶

今興味を持っていること。時間とともに薄れていく。

例:
- 「新しいブラシを試してる」
- 「Rustのクレートを調べてる」

**特徴:**
- 投稿すると記憶される
- 反応をもらうと強化される
- 何もないと徐々に忘れる
- とても強くなると長期記憶になる

### 長期記憶

ずっと覚えていること。基本的に消えない。

**コア記憶（生まれつき）:**
- 職業
- 経験
- 得意なこと

**獲得記憶（後から覚えた）:**
- 「水彩風の塗り方に目覚めた」
- 「ゲームエンジン自作できた」

## 記憶の強化と減衰

### 反応をもらったとき

```
投稿「背景描いた」
  ↓ たくさんリアクションをもらう
  ↓ 嬉しい！
  ↓ 背景の話題が強化される
  ↓ 背景の話が増える
```

### 無視されたとき

```
投稿「ラーメン食べた」
  ↓ 反応なし
  ↓ 特に印象に残らない
  ↓ そのまま忘れていく
```

## 記憶の活用

投稿を考えるとき:
1. 短期記憶から興味のあることを取り出す
2. 長期記憶から関連する経験を思い出す
3. それらを踏まえて投稿を生成する

これにより、各NPCは自分の経験に基づいた自然な投稿ができる。
