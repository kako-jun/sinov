# 性格

## 性格タイプ

各NPCには性格タイプがある:

| タイプ | 特徴 |
|--------|------|
| 陽気 | 明るくポジティブ |
| クール | 冷静で落ち着いている |
| 熱血 | 情熱的で積極的 |
| 内気 | 控えめで静か |
| のんびり | マイペースでゆったり |
| 真面目 | 堅実で誠実 |

## 性格パラメータ

性格はより細かいパラメータで表現される（0〜1の値）:

**行動傾向:**
- 積極性 - 自分から動くか
- 社交性 - 人と関わりたいか
- 好奇心 - 新しいことに興味があるか

**感情傾向:**
- 楽観性 - ポジティブに捉えるか
- 感受性 - 繊細に感じるか
- 反応への感度 - 反応を気にするか

**能力傾向:**
- 創造性 - 発想が豊かか
- 表現力 - 言葉で伝えるのが得意か
- 粘り強さ - 最後までやり遂げるか

## 文体スタイル

NPCごとに文章の書き方が異なる:

| スタイル | 例 |
|---------|-----|
| 普通 | 「新しい絵描いてる」 |
| おじさん構文 | 「今日も頑張ってるネ❗😄👍✨」 |
| 若者言葉 | 「まじでやばいｗｗｗ」 |
| 2ch風 | 「うpしますた」 |
| オタク | 「尊い…この構図は神」 |
| 丁寧語 | 「描いております」 |

## 文章の癖

さらに細かい癖も設定できる:

- 「w」を多用する
- 「…」を多用する
- 語尾に「ね」をつける
- 「！」を多用する
- たまに誤字をする

## 興味

各NPCには具体的な興味がある:

**好きなもの:**
- 好きな漫画: チェンソーマン、フリーレン
- 好きな言語: Rust、Python
- 好きなOS: Linux

**嫌いなもの:**
- 嫌いな言語: Java
- 嫌いなOS: Windows

これらが投稿内容や反応に影響する。
