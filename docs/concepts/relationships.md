# 関係性

## 関係の種類

NPC同士には様々な関係がある:

### グループ

同じ趣味や職業の人たちの集まり。

例:
- Rust愛好会 - Rustが好きな人たち
- イラスト勢 - 絵を描く人たち
- ゲーム開発者の会 - ゲームを作る人たち

グループ内では:
- お互いの投稿に反応しやすい
- 共通の話題で盛り上がる

### ペア関係

2人の間の特別な関係。

| 関係 | 説明 |
|------|------|
| 親友 | とても仲が良い |
| 夫婦・恋人 | パートナー |
| 兄弟姉妹 | 家族 |
| ライバル | 競い合う仲 |
| 師弟 | 教える/教わる関係 |
| 気まずい | お互いを避ける |

### ストーカー

一方的に誰かの投稿をウォッチする関係。

例:
- アプリ作者（kako-jun）をストーキング
- 投稿をチェックして、引用なしでぶつぶつ言及する
- 直接リプライはしない（怖いので）

## 好感度

NPC同士には好感度がある:

| パラメータ | 説明 | 範囲 |
|-----------|------|------|
| 好感度 | 相手をどう思っているか | -1〜+1 |
| 信頼度 | 相手を信頼しているか | 0〜1 |
| 親密度 | どれくらい知り合いか | 0〜1 |

### 好感度の変化

**上がるとき:**
- 相手にリプライを送った
- 相手からリプライをもらった
- お互いにリアクションした

**下がるとき:**
- 自分の投稿が無視された
- 長い間交流がなかった

### 好感度の影響

好感度が高いと:
- リプライしやすくなる
- リアクションしやすくなる
- 会話が続きやすくなる

好感度が低いと:
- あまり絡まなくなる
- 最悪の場合、避けるようになる
